# Todo List APP - FrontEnd Mentor
Projeto realizado com acompanhamento em videoaula do curso DevQuest.

## Stack
- #### **React**
- #### **TaiwlindCSS**
- #### **TypeScript**

## O que aprendi?
Aprendi amplamente a usar **react** com vite na prática, aprendizado inicial em **TailwindCSS**, reforcei meus conhecimentos em **TypeScript**. Este projeto utilizou **ContextAPI** e **Hooks Personalizados** no react.

## Desafio
Após a criação do projeto em videoaula, o professor deixou o desafio de adicionar um botão para deletar uma task, que foi concluído com êxito.
